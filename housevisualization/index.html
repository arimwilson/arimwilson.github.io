Rendering algorithm

Mask editor

Click on the photo to add polygon points. Press Enter to close a polygon and assign it to the active surface. You can draw multiple polygons per surface (e.g. siding panels separated by the garage).

  • Click add point
  • Enter close polygon & save to active surface
  • Esc cancel current polygon
  • Backspace remove last point
  • Right-click a polygon to delete it
  • Scroll zoom · Space + drag to pan

Surface albedos

The real paint color of each surface, used to undo the photo's baked-in lighting so dark→light swaps read correctly. Auto-computed from the brightest diffuse pixels in each surface's masks (95th percentile, to skip specular outliers); use Sample to pick a specific pixel if auto looks off.
